# WallpaperDOOM
WallpaperDOOM is a Python project that turns Chocolate Doom into a live desktop wallpaper by embedding the game behind the desktop icons using the Windows WorkerW window.

Features:

- Runs the original Chocolate Doom as an animated wallpaper.
- Plays behind desktop icons.
- Supports keyboard controls while running as a wallpaper.
- Automatically centers and resizes the game window.

Controls:

- Arrow Keys: Move and turn.
- Ctrl: Fire.
- Alt: Strafe modifier.
- Shift: Run.
- Space: Use/Open doors.
- Tab: Automap.
- Esc: Menu.
- F10: Exit WallpaperDOOM.

Important Notes:

- Mouse input is not supported. Use the keyboard controls listed above.
- WallpaperDOOM should not be used at the same time as Wallpaper Engine or other animated wallpaper applications, as they may interfere with the desktop embedding process.
- Designed for Chocolate Doom on Windows.
- Antivirus software may occasionally warn about the project because it manipulates Windows desktop windows, although it does not perform any malicious actions.

This project is intended for fun and experimentation with the Windows desktop environment and classic DOOM.
